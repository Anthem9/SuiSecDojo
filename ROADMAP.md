# SuiSec Dojo Roadmap

## Completed

- Phase 0: Challenge 01 on-chain loop on Sui testnet.
- Phase 1: Challenge Registry, Profile, Markdown docs, Badge foundation.
- Phase 2: Challenge 01-10, Walrus Site, replay content, CI.
- Phase 3: Public MVP QA, two-wallet isolation, Security Passport foundation.
- Phase 4: No-backend enhancement version with events, leaderboard, static Tutor Mode, local report training, locale switch, and testnet certificate preview.

## Phase 4 Boundaries

- No production backend.
- No database or centralized indexer.
- No real AI Tutor API.
- No mainnet challenge deployment.
- No mainnet certificate mint.
- No report upload or automatic scoring.

## Future Work Requires Separate Design

- Real AI Tutor with guardrails and usage limits.
- Backend/indexer-backed leaderboard.
- Walrus report upload and review flow.
- Mainnet certificate minting.
- Organization or enterprise training routes.
- Full i18n coverage for every long-form document.
